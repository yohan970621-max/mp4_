# Render 배포 방법

이 방법을 사용하면 강사 PC를 계속 켜둘 필요 없이 학생들이 각자 다른 장소와 PC에서 같은 웹주소로 접속할 수 있습니다.

## 준비물

- GitHub 계정
- Render 계정
- 이 프로젝트 압축파일

## 1. GitHub 저장소 만들기

1. GitHub에 로그인합니다.
2. 우측 상단 `+` → `New repository`를 누릅니다.
3. 저장소 이름을 `video-evaluation-web`로 입력합니다.
4. 공개 여부는 `Private`를 권장합니다.
5. 저장소를 생성합니다.

## 2. 프로젝트 파일 올리기

1. 받은 ZIP 압축을 풉니다.
2. GitHub 저장소 화면에서 `uploading an existing file` 또는 `Add file → Upload files`를 누릅니다.
3. 압축파일 자체가 아니라 압축을 푼 폴더 **안의 모든 파일과 폴더**를 업로드합니다.
4. 저장소 최상단에 다음 파일이 바로 보여야 합니다.

```text
render.yaml
package.json
server.js
README_KO.md
```

5. `Commit changes`를 누릅니다.

## 3. Render에 자동 배포

1. Render에 로그인합니다.
2. `New` → `Blueprint`를 선택합니다.
3. 방금 만든 GitHub 저장소를 연결합니다.
4. Render가 저장소 최상단의 `render.yaml`을 읽는지 확인합니다.
5. `Apply` 또는 `Deploy Blueprint`를 누릅니다.
6. 웹서비스와 PostgreSQL 데이터베이스 생성이 끝날 때까지 배포 로그를 확인합니다.

정상 로그의 마지막 부분에는 다음과 비슷한 내용이 나타납니다.

```text
Database migrations completed.
Video Evaluation Web v2.0.0
```

## 4. 최초 설정

배포가 끝나면 Render가 제공하는 주소를 엽니다.

```text
https://video-evaluation-web-xxxx.onrender.com
```

최초 접속 화면에서 다음을 설정합니다.

- 평가 행사명
- 관리자 이름과 PIN
- 강사 이름과 PIN
- 학생 명단과 개인 PIN

PIN은 숫자 4~12자리이며 모든 계정에서 중복되면 안 됩니다.

## 5. 학생들에게 전달

학생들에게 다음 두 가지를 전달합니다.

- Render 웹주소
- 각자의 개인 PIN

학생은 로그인 후 `내 작품` 메뉴에서 YouTube 일부공개 링크를 제출합니다.

## 6. 평가 시작

관리자 화면에서 모든 작품이 제출됐는지 확인한 다음:

1. 제출 상태를 `제출 마감`으로 설정
2. 평가 상태를 `평가 진행`으로 설정
3. 설정 저장

평가 시작 후 학생과 강사가 작품 목록에서 원하는 영상을 자유롭게 시청합니다.

## 7. 결과 확인

관리자 화면에서 다음을 확인합니다.

- 최종 1·2·3위
- 전체 평가자 평균 시청시간
- 클릭자 평균 시청시간
- 평균 시청률
- 유효 시청자 수
- 클릭률
- 완주율
- 작품별 시청자 기록
- 개인별 가장 오래 본 작품

`CSV 내려받기`를 누르면 Excel에서 열 수 있는 결과 파일이 저장됩니다.

## 무료 Render 사용 시 중요 사항

Render의 무료 PostgreSQL 데이터베이스는 생성 후 30일 제한이 있습니다. 한 번의 단기 과정이나 평가에는 사용할 수 있지만, 30일 이상 데이터를 유지하려면 데이터베이스를 유료 플랜으로 올리거나 외부 PostgreSQL 서비스로 이전해야 합니다.

무료 웹서비스는 사용하지 않을 때 절전 상태로 들어갈 수 있어 첫 접속이 느릴 수 있습니다. 수업 시작 전에 관리자가 한 번 접속해 깨워두는 것이 좋습니다.

## 배포 실패 시 확인

### `DATABASE_URL` 오류

Render Blueprint에 `video-evaluation-db` 데이터베이스가 함께 생성됐는지 확인합니다.

### 빌드 중 npm 오류

Render의 배포 로그에서 Node 버전이 24.17.0인지 확인합니다. `render.yaml`에 이미 지정되어 있습니다.

### 화면은 열리지만 영상이 재생되지 않음

- YouTube 영상이 비공개가 아닌지 확인
- 일부공개인지 확인
- 제출 화면 미리보기에서 재생되는지 확인
- 외부 삽입 제한 또는 연령 제한이 없는지 확인

### 학생이 YouTube에서 직접 본 시간이 기록되지 않음

정상입니다. 평가 사이트 내부 플레이어에서 본 시간만 기록합니다.

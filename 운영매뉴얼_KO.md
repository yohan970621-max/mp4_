# 강사용 운영 매뉴얼

## 평가 전날

1. 웹사이트에 관리자 PIN으로 로그인합니다.
2. 계정 관리에서 학생 21명의 이름이 맞는지 확인합니다.
3. 학생들에게 웹주소와 개인 PIN을 전달합니다.
4. 제출 상태를 `제출 가능`, 평가 상태를 `평가 대기`로 둡니다.
5. 학생들에게 YouTube 일부공개 업로드와 링크 제출을 안내합니다.

## 제출 확인

관리자 화면의 `작품 제출 현황`에서 다음을 확인합니다.

- 학생별 작품이 하나씩 있는지
- YouTube ID가 중복되지 않는지
- 영상 길이가 10분 내외인지
- 제출 화면 미리보기에서 실제 재생되는지

영상 길이는 누군가 사이트 안에서 처음 재생하면 자동으로 확인됩니다. 평가 전에 관리자가 모든 영상을 잠깐 열어 재생해도 됩니다.

## 평가 시작

1. 제출 상태를 `제출 마감`으로 변경합니다.
2. 평가 상태를 `평가 진행`으로 변경합니다.
3. 학생과 강사 모두 자신의 PIN으로 로그인합니다.
4. 작품 목록에서 원하는 작품을 자유롭게 선택합니다.

## 권장 평가 시간

작품이 21개이고 각 영상이 약 10분이면 모든 작품을 끝까지 보기에는 시간이 매우 오래 걸립니다. 자유 선택형 평가라면 학생들이 충분히 여러 작품을 탐색할 수 있도록 최소 60~90분 정도의 기간을 주는 편이 낫습니다. 반드시 한 교시 안에서 끝낼 필요는 없습니다.

## 결과 해석

### 전체 평균 시청시간

클릭하지 않은 평가자를 0초로 포함합니다. 많은 사람이 선택하고 오래 본 영상이 유리합니다.

### 클릭자 평균 시청시간

최소 유효시간 이상 본 사람들만 대상으로 평균을 냅니다. 클릭한 뒤 영상이 얼마나 오래 유지됐는지 봅니다.

### 클릭률

평가 가능한 전체 인원 중 한 번이라도 재생한 사람의 비율입니다.

### 평균 시청률

영상 길이 대비 평균적으로 얼마나 시청했는지 나타냅니다. 작품 길이 차이가 크면 이 값을 순위 기준으로 사용하는 것이 더 공정합니다.

### 완주율

유효 시청자 중 영상의 90% 이상을 본 사람 비율입니다.

## 평가 종료

1. 평가 상태를 `평가 대기`로 바꿉니다.
2. 순위와 세부 기록을 확인합니다.
3. CSV를 내려받아 보관합니다.
4. 수상자를 발표합니다.

## 재평가 또는 다음 기수

`시청기록 전체 삭제`는 계정과 작품은 유지하고 시청기록과 노출 순서만 초기화합니다.

완전히 새 기수를 운영하려면 기존 작품을 삭제한 뒤 학생 계정을 개별 수정하거나, 모든 작품과 기록이 없는 상태에서 `학생 명단 일괄 교체`를 사용합니다.
